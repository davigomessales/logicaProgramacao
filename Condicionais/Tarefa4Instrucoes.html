Dia de Corrida

A corrida anual SENAC PE está para começar! Este ano, teremos muitos participantes. Você foi contratado para escrever um programa para fazer as inscrições dos corredores para a corrida e dar instruções a eles no dia da corrida.

Por exemplo, como uma linha do tempo, a inscrição deve seguir essa estrutura.

Veja como funciona nosso cadastro. Existem corredores adultos (maiores de 18 anos) e corredores juvenis (menores de 18 anos). Eles podem se inscrever em um horário mais cedo ou mais tarde. Os corredores recebem um número de corrida e hora de início com base em sua idade e inscrição.

Número da corrida:

  • Os primeiros adultos recebem um número de corrida igual ou superior a 1000.
  • Todos os outros recebem um número abaixo de 1000.

Hora de início:

  • Os inscritos adultos correm às 9h30 ou 11h.
    • Os adultos que se inscreveram cedo correm às 9h30.
    • Os adultos que se inscreveram tarde correm às 9h30.
  • Os inscritos juvenis correm às 12h30 (independentemente da inscrição).

Mas não planejamos corredores com exatamente 18 anos! Nós vamos lidar com isso até o final do projeto.



  1. Os números das corridas são atribuídos aleatoriamente. Fornecemos o código necessário na parte superior do arquivo.

    Para fazer com que esse conjunto de números varie de 0(inclusivo) a 1000(exclusivo), podemos multiplicar o valor retornado por 1000.

    Por fim, para garantir que tenhamos apenas números inteiros de 0 a 999, podemos arredondar para baixo usando Math.floor().
  2. Crie uma variável que indique se um corredor se inscreveu cedo ou não.
    Atribua um valor booleano a esta variável. Você mudará isso depois ao testar diferentes condições do corredor.
  3. Crie uma variável para a idade do corredor e atribua um número a esta variável.
    Você mudará isso mais tarde ao testar diferentes condições do corredor.
  4. Crie uma instrução condicional que verifique se o corredor é adulto E se inscreveu cedo.
    Some 1000 e atribua a raceNumber se isso for verdade.
  5. Crie uma instrução condicional separada abaixo da primeira (começando if novamente). Esta declaração verificará a idade e a hora de inscrição para determinar o horário da corrida.
    Para corredores com mais de 18 anos que se inscreveram cedo, exiba uma mensagem no console informando que eles correrão às 9h30 incluindo o seu número de corrida.
  6. “Adultos que se inscreveram tarde ​​correm às 11h”

    Como já verificamos os adultos que chegaram cedo, podemos escrever uma mensagem como esta:se não, se o corredor tiver mais de 18 anos E não se inscrever cedo, ele correrá às 11h

    Escreva a declaração else if correspondente. Dentro disso, exiba uma string no console informando que eles vão correr às 11 incluindo o seu número de corrida.
  7. “Os inscritos jovens correm às 12h30 (independente da inscrição)”

    Para corredores menores de 18 anos, exiba uma mensagem no console informando que eles correrão às 12h30 incluindo o seu número de corrida.
